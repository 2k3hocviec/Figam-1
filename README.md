# Figam Landing Page

## 📌 Introduction

This project is a **landing page website** built using **HTML, CSS, and JavaScript**.
The website introduces the **Figam platform**, showcasing its interface, features, and design style.

The goal of this project is to practice **frontend development**, layout design, and building a modern landing page UI.

🌐 **Live Demo:**
https://2k3hocviec.github.io/Figam-1/

---

## 🚀 Features

* Responsive landing page layout
* Modern UI design
* Smooth animations and transitions
* Hero section with product introduction
* Feature sections presenting product capabilities
* Call-to-action (CTA) buttons
* Clean and simple navigation structure

---

## 🛠 Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **JavaScript** – Interactive functionality

These technologies are commonly used for building modern frontend interfaces and web applications. ([Cao đẳng Anh Quốc BTEC FPT][1])
